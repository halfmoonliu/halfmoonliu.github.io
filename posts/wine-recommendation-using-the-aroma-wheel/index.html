How can we describe the flavor of wine using huge amount of review data and make recommendation for consumers?

Dataset üîó

The dataset used for this project contains more than 130,000 wine reviews form the Wine Enthusiast website (https://www.winemag.com/). The data is preprocessed and put on kaggle. Besides the reviews writtent by sommeliers on different wine, it also contains columns like country, designation, points, price, province, region, title, variety, winery and so on.

The Aroma Wheel üîó

Aroma words, taken from modified version of wine aroma wheel (Nobel et al., 1987), was used to describe wine from different regions of France. Here are some examples words in the aroma wheel: iris, jasmine, rose (flower) , apple, pear, peach(tree fruit), grass, tomato(vegetable), butter, cream, lager(microbial).

Use Aroma Word Count To Represent Wine Flavor üîó

In this project, I used wine from different French r√©gions to demostrate how a group of wine can be represented using aroma words. The detail steps are as follows:

  1. All wine reviews were tokenized, with stop words and punctuation removed.
  2. Aroma terms were extracted from wine reviews and counted by r√©gions.
  3. Each aroma term count was divided by total aroma word count by r√©gion. The wine from any r√©gion can thus be represented with AromaA 5%, AromaB 3%…adding to 100%.
  4. The similarity of wine from different r√©gion can be calculated using L2 distance between aroma-term-count percentage and thus make recommendations(e.g. wine with similar or different aroma).
